Overview

BallSim is a physics simulation platform that allows you to create, share, and explore interactive ball physics simulations. This documentation provides comprehensive information about the BallSim API, object types, event system, and more.

Whether you're a beginner looking to create your first simulation or an advanced user diving into custom physics rules, this documentation will help you make the most of BallSim's capabilities.

If you're new to BallSim, we recommend starting with the Getting Started section, followed by the Your First Simulation guide.

Basic Concepts

Before diving into creating simulations, it's helpful to understand some basic concepts used in BallSim:

Simulation

A simulation is the overall container for your physics scene. It manages the physics engine, rendering, and object interactions.

Objects

Physical elements in your simulation, such as balls, containers, and zones. Each object has properties that define its behavior.

Physics Parameters

Settings that control how objects interact, including gravity, friction, elasticity, and more.

Events

Triggers that occur during simulation, such as collisions, which can be used to create interactive behaviors.

Your First Simulation

Let's create a simple bouncing ball simulation using the Code Editor. This example demonstrates the basic structure of a BallSim simulation:

// Initialize a new simulation
const simulation = new BallSim.Simulation({
  width: 800,
  height: 600,
  gravity: 9.8,
  friction: 0.1
});

// Create a container (walls)
const walls = simulation.createContainer({
  type: 'rectangle',
  width: 800,
  height: 600,
  elasticity: 0.8
});

// Create a ball
const ball = simulation.createBall({
  radius: 30,
  position: { x: 400, y: 100 },
  velocity: { x: 2, y: 0 },
  color: '#4e54c8',
  elasticity: 0.9
});

// Start the simulation
simulation.start();

This code creates a simple simulation with a single ball bouncing within a rectangular container. The ball will fall due to gravity and bounce off the walls with slight energy loss (elasticity less than 1).

API Reference

The BallSim API provides a comprehensive set of classes and methods for creating and controlling physics simulations.

Simulation Class

The Simulation class is the main entry point for creating and managing simulations.

Constructor

new BallSim.Simulation(options)

Creates a new simulation instance with the specified options.

Parameters:
Name Type Description Default
options.width Number Width of the simulation area in pixels 800
options.height Number Height of the simulation area in pixels 600
options.gravity Number Gravity force (positive values point downward) 9.8
options.friction Number Global friction coefficient (0-1) 0.1
options.timeScale Number Simulation speed multiplier 1.0
Example:
const simulation = new BallSim.Simulation({
  width: 1000,
  height: 800,
  gravity: 9.8,
  friction: 0.2,
  timeScale: 1.5
});

Methods

simulation.start()

Starts the simulation.

simulation.pause()

Pauses the simulation.

simulation.reset()

Resets the simulation to its initial state.

simulation.createBall(options)

Creates a new ball object and adds it to the simulation.

See Ball Object for options.

Returns: Ball object

simulation.createContainer(options)

Creates a new container object and adds it to the simulation.

See Container Object for options.

Returns: Container object

simulation.createZone(options)

Creates a new zone object and adds it to the simulation.

See Zone Object for options.

Returns: Zone object

Physics Engine

The physics engine handles all calculations related to object movement, collisions, and forces.

Properties

simulation.physics.gravity

The gravity vector applied to all objects in the simulation.

Type: Object { x: Number, y: Number }

simulation.physics.friction

The global friction coefficient applied to all objects in the simulation.

Type: Number (0-1)

Methods

simulation.physics.setGravity(x, y)

Sets the gravity vector for the simulation.

x: Number - The x component of gravity vector

y: Number - The y component of gravity vector

simulation.physics.applyForce(object, forceX, forceY)

Applies a force to a specific object.

object: Object - The object to apply force to

forceX: Number - The x component of the force

forceY: Number - The y component of the force

Rendering

The rendering system handles drawing objects to the screen and visual effects.

For detailed information on customizing visual effects, see the Visual Effects and Trails tutorial.
